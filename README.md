
# WayPeak

### WayPeak - это онлайн-сервис для создания множества списков задач.

В данном сайте использовался:

- Vue
- Vuex
- Vue Router
- Vuelidate
- CSS3
- Ванильный JS
- Firebase

Также сайт полностью адаптирован под любое устройство, что делает его ещё более доступным.

## Для установки проекта
```
npm install
```
## Для запуска проекта
```
npm run serve
```
